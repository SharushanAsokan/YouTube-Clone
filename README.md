# YouTube-Clone
Simple one-page website replicating a YouTube Homepage clone, built using only HTML and CSS

This project is a front-end clone of the YouTube homepage layout. It recreates the UI structure including the navigation bar, sidebar, and video content grid using HTML and CSS.
## Features
- Responsive grid layout for video thumbnails
- Fixed header and sidebar layout
- Multi-line text truncation for video titles
- Clickable video thumbnails and channel icons
- Google Fonts integration for typography

## Technologies Used
- HTML5
- CSS3 
- Google Fonts

## How to Run
1. Clone this repository
2. Install Live Server extension
3. Open `index.html` with Liver server in any browser
